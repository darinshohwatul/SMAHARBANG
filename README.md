# SMAHARBANG

# Website Sekolah Berbasis PHP

SMA Harapan Bangsa atau SMAHARBANG adalah **project website sekolah** yang dikembangkan sebagai **tugas akhir semester** oleh kelompok yang terdiri dari 4 orang mahasiswa.  
Website ini dirancang untuk menampilkan informasi umum tentang sekolah serta menyediakan layanan digital seperti **PPDB online**, **E-Learning**, dan **E-Perpustakaan**.


## Tim Pengembang

1. Angel
2. Aulia
3. Darin
4. Putri


## Deskripsi Proyek
SMAHARBANG bertujuan untuk menghadirkan sistem informasi sekolah yang modern dan mudah diakses oleh siswa, guru, dan masyarakat umum.  
Melalui website ini, pengunjung dapat melihat profil sekolah, melakukan pendaftaran siswa baru (PPDB), serta mengakses sistem pembelajaran dan perpustakaan digital.

## Fitur Utama

### 1. **Beranda (Home)**
Menampilkan informasi utama sekolah, pengumuman, dan tampilan umum yang menarik.

### 2. **Profil Sekolah**
Berisi:
- Visi dan Misi Sekolah  
- Daftar Guru dan Tenaga Pendidik  
- Tentang Kami (Sejarah dan Identitas Sekolah)

### 3. **PPDB Online**
Fitur untuk pendaftaran siswa baru secara daring.  
Calon siswa dapat mengisi formulir dan mencetak bukti pendaftaran langsung dari website.

### 4. **Studi**
- **E-Learning** → Terintegrasi dengan *Moodle* untuk pembelajaran daring.  
- **E-Perpustakaan** → Terintegrasi dengan *SLiMS 9 (Bulian)* untuk sistem peminjaman buku digital.

### 5. **Ekstrakurikuler**
Menampilkan berbagai kegiatan ekstrakurikuler yang ada di sekolah.

### 6. **Fasilitas Sekolah**
Informasi dan galeri tentang fasilitas penunjang kegiatan belajar mengajar di sekolah.


## Teknologi yang Digunakan
Project ini dibangun menggunakan:
- **PHP** — sebagai bahasa pemrograman utama untuk sisi backend  
- **HTML** — untuk struktur halaman  
- **CSS** — untuk tampilan dan desain antarmuka  
- **JavaScript** — untuk interaktivitas dan fungsi dinamis  
- **Moodle** — sebagai platform e-learning  
- **SLiMS 9 (Bulian)** — sebagai sistem e-perpustakaan


## Cara Menjalankan Project
1. Clone repository ini ke folder `htdocs` (jika pakai XAMPP) atau `www` (jika pakai Laragon):
   ```bash

2. Jalankan server lokal kamu (XAMPP/Laragon).

4. Akses project melalui browser:
